# Maven Market Semantic Model
## Fact Table
- Transaction_Data

## Dimensions
- Calendar
- Customers
- Products
- Regions
- Stores
- Return_Data

## Business purpose

This semantic model analyses retail sales performance.
Main KPIs:
- Transactions
- Revenue
- Profit
- Profit Margin
- Returns

Business questions:

- Why did profit decrease?
- Which stores perform best?
- Which products generate the highest profit?
- Which regions are underperforming?
- What trends exist over time?

The AI should answer questions using these business entities.
