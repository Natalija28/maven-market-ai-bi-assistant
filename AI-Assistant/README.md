# Power BI AI Business Analyst
## Goal
Build an AI Business Analyst that understands Power BI semantic models, explains dashboards, generates DAX, SQL and Power Query, reviews PBIP projects and helps users analyse business performance using AI.
## Features
 - Explain DAX measures
 - Generate DAX
 - Generate SQL
 - Fenerate Power Query (M)
 - Explain Power BI models
 - Review PBIP projects
 - Connect to OpenAI API
 - Create documentation
 - Suggest dashboard improvements
